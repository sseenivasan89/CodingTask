# CodingTask

## BDD Cucumber with Page Object Model

**Clone the project Directory**

## Command Run the Project

```
mvn clean verify
```

> Dependencies
>  - selenium-java
> - io.cucumber
> - TestNg
> - JUnit

> Highlights
> - Hooks
> - Page Object Model
> - Screenshot on failure - \FailedScreenshots
> - Maven Cucumber Reporting - \target\cucumber-report-html\cucumber-html-reports
